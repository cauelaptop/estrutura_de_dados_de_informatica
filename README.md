# LEIA ESTE TEXTO ANTES DE CONTINUAR

Este projeto em HTML tem como objetivo ajudar estudantes da área de informática, oferecendo materiais variados para auxiliar nos estudos, além de servir como apoio para profissionais ou iniciantes que desejam aprender o básico sobre tecnologia e informática.

O site ainda está em desenvolvimento, mas a proposta do projeto é criar um material completo contendo:

* Linguagens de Programação
* Informações e histórias sobre Hardwares
* Informações e tutoriais de Sistemas Operacionais
* Exercícios práticos para desenvolver conhecimentos em informática

O HTML principal (**index.html**) foi desenvolvido por **Cauê Shishido** e funciona como o menu principal do site, permitindo que o usuário escolha sua área de estudo e navegue pelo conteúdo.

Já os HTMLs secundários (**index/nome-da-opcao.html**) foram desenvolvidos por **Lucas Fernandes** e **David Delgado**, com o objetivo de proporcionar uma experiência de aprendizado prática e acessível para os usuários.
